# RoadTwin AI — Predictive Road &amp; Asset Management Platform

An interactive, animated HTML presentation for an AI-powered road / pavement and roadside-asset management platform, built for **Traffic Tech**.

It captures the live **as-is** condition of every road and roadside asset, fuses it with history, materials science and predictive AI, and drives budget-optimised maintenance decisions.

## Sensing technologies featured
- **XenoTrack** — 6D LiDAR surface scanning (IRI, rutting, PCI, cross-slope, edge-AI distress detection)
- **PaveSense** — acoustic &amp; vibration sub-surface sensing (delamination, voids)
- **EchoTwin** — vision-AI living digital twin for roadside assets

## Deck contents (19 slides)
Vision → platform architecture → the three sensing technologies → sensor-fusion specs → correlation engine → distress taxonomy &amp; condition indices → predictive models → pavement lifecycle economics → decision engine → treatment matrix → budget optimisation → asset catalogue → digital-twin data architecture → deployment roadmap → outcomes.

## Viewing
Open `index.html` in any modern browser, or view the hosted version on GitHub Pages.

**Controls:** `←` / `→` to navigate · `F` for fullscreen.

Single self-contained file — no build step, no external dependencies (the Traffic Tech logo is embedded).
